# React + Vite

Para clonar correctamente el proyecto:
1-Clonar proyecto
2-Abrir terminal
3-Acceder a la carpeta mamma-mia con el comando: cd mamma-mia
4-Luego escribir el siguiente comando: npm install
5-Esperar a que se instalen las librerias faltantes del proyecto
6-Escribir el siguiente comando: npm run dev
7-Listo!:)
